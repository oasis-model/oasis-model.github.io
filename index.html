Oasis: Generating Worlds in Realtime

Etched Team, Decart Team

View codePaper (Coming Soon)

We have trained a new model, Oasis, that can generate 3D worlds in realtime. Oasis builds on the work of Diffusion Forcing, by introducing a new DiT backbone and a ViT-based VAE. In this technical report, we will discuss the model behaviour and results, detail the architecture and training process, and release the code open-source.

Architecture

In contrast to bidirectional models such as Sora, Oasis can generate an unlimited length of video autoregressively, with the option to be conditioned on game input. This enables users to interact with the world in realtime.

Temporal consistency is enforced by a Diffusion Forcing mechanism, which trains the model to be consistent with itself over time. However, we found that over longer sequences (100+ frames) the model would begin to lose consistency.
